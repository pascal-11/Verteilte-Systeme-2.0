Um das Spiel starten und spielen zu können muss folgendermaßen vorgegangen werden:

Installation:

Folgende Befehle müssen im Ordner "pong-endgültigfertig" durchgeführt werden:

Installieren von socket.io mit "nmp install socket.io"

Installieren von Express mit "nmp install express"



Starten:

Folgende Befehle müssen im Ordner "pong-endgültigfertig" durchgeführt werden:

Starten des Servers mit "node server.js"

Browser (Chrome oder Firefox) öffnen und "localhost" für den Port 3000 eingeben

-> Hauptmenü öffnet sich
-> Viel Spaß!
